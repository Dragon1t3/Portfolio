Details

This API is a proto-type list of a number of coffee shops in London.

The API consists of:

  • • Location and directions using Google Maps
  • • Available WiFi access
  • • Coffee prices
  • • Available seats
  • • Restrooms

Users can make location specific GET requests and POST requests (with a valid API-KEY) to make changes to a listed coffee shop, or add a coffee shop if it is not on the list.

Technologies Used

  • Python
  • HTML5
  • CSS3
  • Flask
  • SQLAlchemy
