Details

StudyBuddy is a threading website which allows users to study together and share their knowledge. Each user can create a room for a topic the user wants to discuss, and others can choose to join. All the rooms the user has created or participated in and the message history is easily viewed within the user profile page.

StuddyBuddy feautures consists of:

  • • User-Authentication
  • • User can create new rooms
  • • All users can make a post in a room
  • • Comments can be editing by respective user
  • • User profiles
  • • Profile pics

Technologies Used

  • Python
  • HTML5
  • CSS3
  • DJango
  • JavaScript
  • SQLite3
