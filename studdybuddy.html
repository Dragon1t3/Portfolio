Details

This was one of the most challenging projects I’ve worked on and I was the sole developer on it. Containing one to many and many to one relationships. This really pushed my skills to it’s limits and forced me to do research on certain elements of the project.

StuddyBuddy feautures consists of:

  • • User-Authentication
  • • User can create new rooms
  • • All users can make a post in a room
  • • Comments can be editing by respective user
  • • User profiles
  • • Profile pics

When visiting a users profile you will be able to see all the rooms this user has participated in and there is also a history of the messages the user has sent.

Technologies Used

  • Python
  • HTML5
  • CSS3
  • DJango
  • JavaScript
  • SQLite3
